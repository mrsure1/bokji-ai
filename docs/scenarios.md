# bokji-ai 서비스 시나리오 정의서 (Scenarios)

본 문서는 사용자가 bokji-ai 서비스를 이용하는 실제 시나리오를 정의하여, 화면상에서 보기 좋고 한눈에 파악하기 쉬운 UI/UX가 어떻게 구현되어야 하는지 지침을 제공합니다. 

본 서비스는 문자 및 이메일 알림을 부수적 편의 옵션으로 두되, **모바일 웹앱 화면 내에서 직관적이고 쉬운 정보 시각화를 제공하는 것을 메인 가치**로 삼습니다.

---

## 1. 서비스 기본 사용 흐름 (Flow)

```text
+---------------------------------------------------------+
|                  [ 사용자 자연어 입력 ]                  |
|   예: "30대 취준생이고 원룸 월세가 너무 부담돼요..."      |
+---------------------------------------------------------+
                            │
                            ▼
+---------------------------------------------------------+
|                [ 백엔드 & AI 분석 ]                     |
|  1. 프로필 매칭 (연령, 가구원수, 주거형태, 소득수준)       |
|  2. 벡터 검색 (pgvector) 및 RAG로 최적 혜택 추출          |
|  3. LLM 기반 쉬운 말 요약 및 시각적 데이터 구조화       |
+---------------------------------------------------------+
                            │
                            ▼
+---------------------------------------------------------+
|           [ 메인 기능: 직관적 UI 화면 출력 ]             |
|                                                         |
|  ┌──────────────────────────────────────────────────┐  |
|  │  ★ 청년월세 특별지원 (신청가능성: 높음)          │  |
|  │  - 지원 내용: 월 최대 20만 원 (최대 12개월)        │  |
|  │  - 신청 기간: 2026.12.31 까지 [D-180]            │  |
|  └──────────────────────────────────────────────────┘  |
|  ┌──────────────────────────────────────────────────┐  |
|  │  ★ 청년수당 (확인 필요: 소득 기준)                │  |
|  │  - 지원 내용: 월 50만 원 x 6개월                  │  |
|  └──────────────────────────────────────────────────┘  |
+---------------------------------------------------------+
                            │
                            ▼ (부수적 선택 사항)
+---------------------------------------------------------+
|          [ 부수적 옵션: 문자 / 이메일 알림 ]            |
|  - 마감 임박 알림 수신 (SMS로 중요 일정 안내)            |
|  - 준비서류 목록 및 신청 가이드 발송 (Email로 가이드 제공)|
+---------------------------------------------------------+
```

## 2. 시나리오 1: 30대 취업 준비 1인 가구 여성 (김민지 씨)

### 2.1. 페르소나 및 배경 상황
* **이름 / 나이:** 김민지 (32세, 여성)
* **거주 형태:** 서울시 관악구 소재 원룸 (보증금 1,000만 원 / 월세 50만 원)
* **가구 구성:** 1인 가구 (싱글)
* **경제 상태:** 취업준비생 (현재 무직, 월세 및 고정 지출로 인해 매달 생활비 부족)
* **목적:** 당장 다가오는 월세 부담을 줄이고, 취업 준비 기간 동안 생계 지원을 받을 수 있는 실질적인 정책 찾기

### 2.2. bokji-ai 앱 사용 여정

1. **온보딩 및 간편 프로필 등록**
   * 민지 씨는 복잡한 서류 제출 없이 나이(1994년생), 지역(서울 관악구), 주거(월세), 고용상태(무직/구직중)를 몇 번의 탭만으로 빠르게 설정합니다.
2. **자연어 상황 입력**
   * 채팅 창에 편하게 자신의 사정을 입력합니다: 
     > *"서울 원룸 월세 사는 30대 취준생인데, 매달 월세 내기가 너무 벅차요. 생활비랑 월세 지원받을 수 있는 혜택이 있을까요?"*
3. **한눈에 들어오는 맞춤 혜택 카드 화면 (Main UI)**
   * 복잡한 행정 사이트처럼 글자가 빽빽하지 않고, **주거비 지원**과 **구직 활동 지원** 카테고리가 뚜렷한 색상(예: 파스텔톤 블루 및 그린)으로 구분되어 나타납니다.
   * `신청 가능성: 높음` 태그가 붙은 **서울 청년월세지원** 카드가 가장 위에 배치되며, 지원금액인 **"월 최대 20만 원"**이 굵고 크게 강조됩니다.
   * 신청 마감일까지 남은 날짜(`⏰ D-15`)가 붉은색 배지로 한눈에 보입니다.
4. **쉬운 말 요약 및 체크리스트 확인**
   * 상세 화면을 터치하자 AI가 요약한 **"받을 수 있는 지원"**, **"신청 대상 조건"**이 세 줄 요약으로 깔끔하게 정돈되어 제공됩니다.
   * 복잡한 소득 기준(기준 중위소득 150% 이하 등)을 민지 씨가 이해하기 쉽게 **"내 월 소득 세전 OO만 원 이하이면 가능해요"**라고 직관적인 수치로 환산하여 보여줍니다.
   * 신청에 필요한 임대차계약서, 월세 이체 증빙 서류 등이 **체크박스 목록**으로 표시되어 준비 여부를 터치로 체크해 볼 수 있습니다.
5. **부수적 알림 옵션 설정 (보조 기능)**
   * 당장 서류 준비가 어려워 우선 해당 혜택을 '저장'합니다.
   * 마감 직전에 놓치지 않도록 **"마감 3일 전 문자 알림 받기"** 스위치를 켜서 알림을 설정합니다.

## 3. 시나리오 2: 50대 실직 및 건강 취약 1인 가구 남성 (박영호 씨)

### 3.1. 페르소나 및 배경 상황
* **이름 / 나이:** 박영호 (55세, 남성)
* **거주 형태:** 경기도 수원시 소재 다세대주택 반전세 (보증금 500만 원 / 월세 30만 원)
* **가구 구성:** 1인 가구 (싱글, 자녀 및 배우자 없음)
* **경제 상태:** 최근 회사 경영난으로 권고사직(실직 상태), 통장 잔고 거의 없음
* **건강 상태:** 당뇨 합병증 및 척추 질환으로 인해 정기적 외래 진료 및 약 복용 필요, 병원비 부담 급증
* **목적:** 즉각적인 생계 지원(식비, 월세) 및 병원비를 줄일 수 있는 의료비 감면 혜택 탐색

### 3.2. bokji-ai 앱 사용 여정

1. **온보딩 및 큰 글씨 모드 적용**
   * 시력이 다소 침침한 영호 씨를 위해 첫 화면부터 **"큰 글씨 보기"** 옵션이 눈에 띄게 제공됩니다. 
   * 나이(55세), 지역(경기 수원), 가구원수(1인), 질환(만성질환), 소득수준(무직/재산 없음)을 큼직한 버튼으로 설정합니다.
2. **자연어 상황 입력**
   * 복잡한 검색어를 생각할 필요 없이 음성 인식이나 간단한 텍스트로 속사정을 말하듯 입력합니다:
     > *"얼마 전에 회사 잘려서 돈이 하나도 없고 몸도 많이 아파서 병원 치료를 계속 받아야 하는데 도와줄 복지가 있습니까?"*
3. **가독성 극대화 화면 배치 (Main UI)**
   * 화면에는 생명과 직결된 **의료 지원**과 당장의 생계를 보장하는 **긴급 생계 지원**이 최우선적으로 배치됩니다.
   * 복잡한 자격 요건 표 대신, **"병원비 걱정 줄이기(긴급의료지원)"**, **"생계비 지원(긴급복지)"**처럼 이해하기 쉬운 명칭의 카드 UI로 제공됩니다.
   * 각 카드는 고대비 배색(Dark Mode 또는 높은 명도 대비)으로 디자인되어 시인성이 극대화되어 있습니다.
4. **쉬운 용어 풀이 및 원클릭 신청 가이드**
   * 상세 설명에서 '차상위 본인부담경감', '소득인정액' 같은 어려운 한자어나 행정 용어가 나올 때마다 밑줄이 그어져 있고, 터치 시 **"쉽게 설명한 용어 팝업"**이 나타납니다.
     * *예: "차상위 본인부담경감이란? -> 몸이 아파서 병원에 갈 때 국가가 병원비의 대부분을 대신 내주는 제도예요."*
   * 동 주민센터에 방문해서 말해야 하는 **"핵심 멘트 가이드"**와 **"필수 지참 서류"**를 한눈에 볼 수 있도록 메모지 형태의 UI로 제공합니다.
5. **부수적 알림 및 가이드 공유 설정 (보조 기능)**
   * 영호 씨는 스마트폰으로 긴 글을 읽는 것이 피로하므로, **"필수 서류 목록 및 찾아가는 길 이메일로 보내기"** 버튼을 눌러 본인의 이메일로 가이드를 발송합니다.
   * 혹시 잊어버릴 것에 대비해 매달 추천되는 신규 긴급 지원 혜택에 대한 요약을 **"문자로 받아보기"**로 설정해 둡니다.

## 4. UI/UX 구현을 위한 개발 핵심 지침

1. **정보 밀도 최소화:** 한 화면에 담기는 텍스트 양을 줄이고, 카드 간 간격을 넓혀 시각적 피로를 줄입니다.
2. **명확한 색상 코딩:** 혜택의 카테고리(주거, 구직, 의료, 생계 등)와 신청 가능성 상태(높음, 확인 필요, 불가능)를 보편적이고 구별하기 쉬운 색상 칩과 배지로 구분합니다.
3. **가독성 향상 도구 지원:** 폰트 크기 조절 기능(큰 글씨 모드)과 고대비 테마(접근성)를 기본 탑재합니다.
4. **원터치 액션:** 저장, 알림 설정, 가이드 발송 등의 부수 기능을 상세 화면에서 1회 탭으로 처리 가능하도록 간소화합니다.
